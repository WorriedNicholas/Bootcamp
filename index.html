HTML

  • "h"surrounded in arrow brackets creates a new header. to create multiple headers assign a number in chronological order after h i.e(h1, h2)
  • start an html file with "!DOCTYPE html" surrounded in arrow brackets to let the computer know that this is a html document
  • "li" surrounded in arrow brackets adds a new line of text in an unordered list
  • "ul" in arrow brackets allows us to place "li" inside of "ui"

CSS Notes

  • "margin" and "padding" both effect how much space is between elements
  • you can use a color name or a hex value or rgb values to change the color of text and other items

    Git Notes

  • git is a version control software
  • "git status" in the terminal tells us the current branch we are working in as well if it's up to date
  • "git checkout branch_name" assigns you to that branch
  • "git checkout -b new_branch_name" creates a new branch with that name
  • "git pull origin main" downloads the main branch to ensure you are up to date with the main branch
  • "git push origin branch_name" uploads the current code to that branch
  • "git commit -m "your message here"" adds a new message for changes in the merge on github
  • "git add -A" stages your changes

    JavaScript Notes

  • JavaScript is a programming language that implements back end and features
  • JavaScript is an object oriented(OOP) programing language
  • "function function_name" creates a function
  • we put the code for the function inside curly brackets {}
  • "console.log" displays messages in the chrome developer console, similar to the print() command in python
  • "if" statements are conditional statements that only excute if the condition is met
  • "var variable_name = variable_value" creates and store that value in the variables name
